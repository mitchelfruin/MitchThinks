The Site

I built the site using the R package blogdown for a few reasons:

  • I wanted to learn the (very) basics of web development.
  • I wanted to post book summaries to encourage myself to read (and actually think about what I’ve read).
  • I wanted to use R for something that didn’t feel like work.

The site is split into two sections:

  • Book summaries.
    • Self-contained summaries of interesting non-fiction books, illustrated with R code, with a few of my thoughts thrown in.
  • Interests.
    • Random articles on various topics that I’m interested in and trying to learn more about, including:
      • Programming.
      • Data Science (analytics, statistics, and machine learning).
      • Alternative approaches to economics (e.g. behavioural, complexity).
      • Evolutionary biology.
    • Once complete, each page would hopefully serve as an Intro to X.
